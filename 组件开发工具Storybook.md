# 组件开发工具Storybook

## 简介

StoryBook 是一个开源的 UI 组件库构建工具，支持 React、Vue、Angular 等主流开发框架，使用 StoryBook 将获得以下优势：

    开发环境隔离并以可交互的方式展示组件
    便捷且直观组件分组管理方式
    同样使用 webpack 构建，集成 react-hot-loader 调试方便
    可配置化程度高，插件化，社区完善
